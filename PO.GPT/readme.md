A powerful CLI tool that leverages AI to automatically translate GNU gettext PO/POT files. Say goodbye to manual
translation work!

## Features

- 🤖 **AI-Powered Translation** - Uses OpenAI-compatible APIs (OpenAI, Claude, local models, etc.)
- 📦 **Batch Processing** - Efficiently translates multiple entries in batches
- 🔄 **Smart Merging** - Merges POT templates with existing PO files
- 💰 **Cost Tracking** - Real-time token usage and cost estimation
- 🎯 **Context-Aware** - Passes msgctxt to AI for more accurate translations
- 🧪 **Dry Run Mode** - Test translations without making API calls or modifying files
- ⚡ **Multiple Languages** - Translate to multiple target languages in one go
- 📊 **Beautiful CLI** - Rich terminal UI with progress indicators
